# Projekt biblioteki w Javie

## Opis projektu
Projekt realizuje prostą bibliotekę:
- Przechowuje książki w tablicy
- Pozwala dodawać książki
- Wypożycza i zwraca książki dla czytelników
- Wyświetla dostępne książki i licznik wypożyczeń

## Scenariusz testowy
1. Dodanie książek
2. Wypisanie dostępnych książek
3. Wypożyczenie książki przez czytelnika
4. Ponowne wypisanie książek
5. Zwrot książki
